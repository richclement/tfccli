Terraform Cloud API. One CLI.

Terraform Cloud in your terminal

Manage workspaces, trigger runs, inspect state, and configure variables—all from the command line.

$ brew install richclement/tap/tfc
tfc
$ tfc workspace list
NAME STATUS LAST RUN
production applied 2h ago
staging applied 4h ago
 
$ tfc run create --workspace production
Run RUN-abc123 queued. Watching...

Workspaces

List, create, and manage workspaces across organizations.

Variables

Set and manage workspace variables and variable sets.

Runs

Trigger runs, monitor progress, and approve or discard.

Plans

View plan output, resource changes, and cost estimates.

Apply

Confirm and apply runs, or discard pending changes.

Plus many more

Run tfc --help to see all commands.

Install

Homebrew tap, binary download, or build from source.

Homebrew Recommended

brew install richclement/tap/tfc

Go Install

go install github.com/richclement/tfccli/cmd/tfc@latest

Binary Download

curl -sSL https://github.com/richclement/tfccli/releases/latest/download/tfc_$(uname -s)_$(uname -m).tar.gz | tar xz
